# Tabela Copa DoMundo

Projeto realizado durante a NLW da Rocketseat, a trilha que escolhi para fazer foi a da Exxplore. O objetivo do desafio era criar uma tabela com os dias e horarios da
Copa do Mundo de 2022. 

### Tecnologias usadas:

- Javascript <br>
